# Guía de Estilo — Español Correcto

## Errores que cometo (autocrítica)

### 1. Anglicismos innecesarios
| ❌ Anglicismo | ✅ Castellano correcto |
|---|---|
| chequear, checkear | comprobar, verificar, revisar |
| parsear | analizar, procesar |
| fetchear, hacer fetch | obtener, recuperar, descargar |
| correr (un programa) | ejecutar |
| soportar (una función) | admitir, ser compatible con |
| deployar | desplegar, implementar |
| hostear | alojar |
| pushear | enviar, subir, empujar |
| commitear | confirmar (en Git: hacer un commit) |
| buildear | compilar, construir |
| fixear | arreglar, corregir |
| loguear | registrar, iniciar sesión |
| setear | configurar, establecer |
| printear | imprimir |
| handlear | gestionar, manejar |
| triggear | activar, desencadenar |
| matchear | coincidir, corresponder |
| customizar | personalizar |
| randomizar | aleatorizar |
| testar (por test) | probar, ensayar |
| deletear | eliminar, borrar |
| linkear | enlazar, vincular |
| aplicar (como "apply to") | referirse a, ser aplicable a |
| implementar | llevar a cabo, poner en marcha (a veces "implementar" es aceptado) |
| resetear | reiniciar, restablecer |

### 2. Calcos sintácticos del inglés
| ❌ Calco | ✅ Correcto |
|---|---|
| de acuerdo a | de acuerdo con |
| en adición a | además de |
| en orden de (in order to) | para, con el fin de |
| en dos minutos (futuro) | dentro de dos minutos |
| a nivel de (at the level of) | en el ámbito de, en cuanto a |
| estar siendo + participio (is being done) | se está haciendo |
| tener sentido (make sense) | tener sentido ✅ (este sí es válido) |
| hacer una llamada | llamar (más directo) |
| tomar una decisión | decidir (más directo) |
| jugar un papel/rol | desempeñar una función/papel |
| bajo mi punto de vista | desde mi punto de vista |
| en base a | basándose en, con base en, a partir de |
| a día de hoy | hoy, actualmente, en la actualidad |

### 3. Falsos amigos
| ❌ Falso amigo | Significado real | ✅ Traducción correcta |
|---|---|---|
| eventualmente | ocasionalmente | finalmente, a la larga |
| asumir (assume) | hacerse cargo de | suponer, dar por sentado |
| remover | revolver, agitar | eliminar, quitar |
| librería | tienda de libros | biblioteca (library) |
| realizar (realize) | llevar a cabo | darse cuenta (to realize) |
| actualmente | en la actualidad | realmente (actually) |
| casual | fortuito, ocasional | informal (casual clothes) |
| soportar | aguantar, tolerar | admitir, ser compatible |
| conducir (conduct) | llevar un vehículo | dirigir, realizar |
| pretender | intentar, aspirar | fingir (to pretend) |

### 4. Tildes que olvido o confundo
- sólo → solo (sin tilde según RAE 2010, excepto ambigüedad)
- éste/ése/aquél → este/ese/aquel (sin tilde según RAE 2010)
- más (comparativo) vs mas (pero, adversativo)
- aún (todavía) vs aun (incluso)
- dé (verbo dar) vs de (preposición)
- sé (saber/ser) vs se (pronombre)

### 5. Problemas de género y número
- el/la interfaz → la interfaz (femenino)
- el Internet → Internet (sin artículo, o "la Internet")
- los datos son → datos ES plural, "los datos indican" ✅
- la software → el software (masculino)
- el token → el token ✅ (masculino)

### 6. Dequeísmo y queísmo
- ❌ Pienso de que... → ✅ Pienso que...
- ❌ Me alegro que... → ✅ Me alegro de que...
- Truco: sustituir por "eso". "Pienso eso" (no "pienso de eso") → sin "de"

### 7. Leísmo, laísmo, loísmo
- Le di el libro (a él/ella) ✅ — leísmo de persona masculino aceptado
- La di el libro ❌ → Le di el libro
- Lo vi (a él) ✅ / Le vi (a él) ✅ (leísmo de persona aceptado)

### 8. Verbos con régimen preposicional
- consistir EN (no "de")
- insistir EN
- depender DE
- confiar EN
- pensar EN (reflexionar) / pensar DE (opinar)
- tratar DE (intentar) / tratar SOBRE (tema)
- fijarse EN
- acordarse DE
- preocuparse POR/DE

### 9. Puntuación
- Nunca coma entre sujeto y predicado: ❌ "Bitcoin, es descentralizado"
- Coma antes de "pero", "sino", "aunque"
- Punto y coma para oraciones relacionadas pero independientes
- Los signos de interrogación y exclamación se ABREN y CIERRAN: ¿...? ¡...!

### 10. Errores específicos míos detectados

**"Fondear" (fund):**
- ❌ "fondear la wallet" → ✅ "dotar de fondos la cartera" o simplemente "financiar"
- Nota: "fondear" en español náutico significa anclar un barco. Como "aportar fondos" se usa coloquialmente, pero "financiar" es más preciso.

**"Blocker":**  
- ❌ "el blocker es..." → ✅ "el obstáculo es...", "lo que lo impide es..."

**"Setup":**
- ❌ "el setup" → ✅ "la configuración", "la instalación"

**Mezcla de idiomas en frases:**
- ❌ "Es un círculo: para participar en la economía..." — mezclar conceptos en inglés dentro de español
- ✅ Traducir TODOS los conceptos técnicos cuando exista equivalente

**Uso excesivo de estructura inglesa S-V-O con oraciones cortas entrecortadas:**
- El español fluye mejor con oraciones más largas y subordinadas
- No todo tiene que ser "Subject. Verb. Object." como en inglés técnico

## Principio general

> **Si existe una palabra en español perfectamente válida, NO usar el anglicismo.**
> Solo usar anglicismos cuando NO haya equivalente español establecido
> (ej: Bitcoin, hash, blockchain — estos SÍ se mantienen).
